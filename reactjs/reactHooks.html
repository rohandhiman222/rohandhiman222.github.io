# Hooks

# What are hooks?

Hooks are a new features addition in React version 16.8 which allow you to use React features without having to write a class. Ex: State of a component.

WARNING

Hooks don't work inside classes

# Why Hooks?

Reason One

  • Understand how this keyword works in javascript
  • Remember to bind event handlers in class components
  • Classes don't minify very well and make hot reloading very unreliable.

Reason two

  • There is no particular way to reuse stateful component logic.
  • HOC and render props patterns do address this problem.
  • Makes the code harder to follow
  • There is need a to share stateful logic in a better way.

Reason Three

  • Create components for complex scenarios such as data fetching and subscribing to events Related code is not organized in one place.

Ex: Data Listeners -> in componentDidMount and componentWillUnmount

  • Because of stateful logic - can't break component into smaller ones

Rules of Hooks

  • Only call hooks at the top level
  • Don't call hooks inside loops, conditions, or nested function.
  • Only call hooks from react functions.

Call them from within react functional components and just any regular Javascript function.

# component state update automatic update title

import React, { Component } from "react";

export default class ClassCounterTitle extends Component {
  constructor(props) {
    super(props);
    this.state = {
      count: 0,
    };
  }

  componentDidMount() {
    document.title = `Clicked ${this.state.count} times`;
  }

// automatic call
  componentDidUpdate(prevProps, prevState) {
    document.title = `Clicked ${this.state.count} times`;
  }

  render() {
    return (
      <div>
        <button onClick={() => this.setState({ count: this.state.count + 1 })}>
          Click {this.state.count}
        </button>
      </div>
    );
  }
}

// compare previous state and current state 

import React, { Component } from "react";

export default class ClassCounterTitle extends Component {
  constructor(props) {
    super(props);
    this.state = {
      count: 0,
      name: "",
    };
  }

  componentDidMount() {
    document.title = `Clicked ${this.state.count} times`;
  }

  componentDidUpdate(prevProps, prevState) {
    if (prevState.count !== this.state.count) {
      console.log("workign ");
      document.title = `Clicked ${this.state.count} times`;
    }
  }

  render() {
    return (
      <div>
        <input
          type="text"
          onChange={(e) => {
            this.setState({ name: e.target.value });
          }}
        />
        <button onClick={() => this.setState({ count: this.state.count + 1 })}>
          Click {this.state.count}
        </button>
      </div>
    );
  }
}
