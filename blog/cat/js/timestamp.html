# 1. Disable Timestamps

Laravel by default provides created_at and updated_at. If your DB doesn't have these fields. You will get an SQL error. If you don't need these fields we can disable them. go to your model class.

class StudentInfo extends Model
{
    public $timestamps = FALSE;
}

# 2. Change timestamp column names

If you have a different name column of your Database we also have to change Laravel. I mean, both should be the same otherwise, we will get an SQL error. We have create_time and update_time in our DB we have to change as well in Laravel.

class StudentInfo extends Model
{
    const CREATED_AT = 'create_time';
    const UPDATED_AT = 'update_time';

}
