                  # What is this thing called scope?

                  • A scope is a certain region of a program where a defined variable exists and can be recognized and beyond that it cann't be recognized. so there can be multiple types of scopes for example

                  • Global scope

                  • Block scope

                  • Function scope

                  Var

                  Var is function scope

                  Let and Const

                  Let and const both are block scope.

                  var a = "rohan"; // global scope

function name() {} // function scope

{
} // block scope

                  # Variable Shadowing

                  function test() {
  let a = "hello";
  if (true) {
    let a = "hey"; // this a is shadow
    console.log(a);
  }
  console.log(a);
}

// no error output both are different
// hello
// hey

                  # Illegal Shadowing

                  // error
function test() {
  var a = "hello"; // here you can use var variable
  let b = "bye";
  if (true) {
    let a = "hey";
    var b = "Goodbye"; // error you can't do with var variable
    // b has been already declared, this is called Illegal shadowing
    console.log(a);
  }
  console.log(a);
}
