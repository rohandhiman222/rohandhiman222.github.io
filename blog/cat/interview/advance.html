                                                                              # JavaScript Promises: A Comprehensive Guide

                                                                              Promises are a key feature in JavaScript for handling asynchronous operations, providing a more manageable approach than callbacks.

                                                                              # Creating a Promise

                                                                              A new promise is created using the Promise constructor:

                                                                              const myPromise = new Promise((resolve, reject) => {
    // Asynchronous operation code
    if (/* operation successful */) {
        resolve(value);
    } else {
        reject(error);
    }
});

                                                                              # Consuming a Promise

                                                                              To act on the resolution or rejection of a promise, use the .then(), .catch(), and .finally() methods.

                                                                              # .then()

                                                                              The .then() method is used for scheduling a callback to be executed when the promise is resolved. It takes up to two arguments: the first is a callback function for the resolved case of the promise, and the second (optional) is a callback for the rejected case.

                                                                              myPromise.then(
    value => { /* handle a resolved value */ },
    error => { /* handle an error (optional) */ }
);

                                                                              .catch()

                                                                              Handles rejected cases only.

                                                                              myPromise.catch(
    error => { /* handle an error */ }
);

                                                                              .finally()

                                                                              Executes a callback when the promise is settled, regardless of its outcome.

                                                                              myPromise.finally(
    () => { /* code to run after the promise is settled */ }
);

                                                                              # Static Methods of Promise

                                                                              # Promise.resolve()

                                                                              Returns a Promise object that is resolved with a given value.

                                                                              const resolvedPromise = Promise.resolve(value);

                                                                              Promise.reject()

                                                                              Returns a Promise object that is rejected with a given reason.

                                                                              const rejectedPromise = Promise.reject(error);

                                                                              Promise.all()

                                                                              Takes an iterable of promises and returns a single Promise that resolves to an array of the results.

                                                                              Promise.all([promise1, promise2, promise3])
  .then(values => {
    console.log(values);
  });

                                                                              Promise.allSettled()

                                                                              Returns a promise that resolves after all of the given promises have either resolved or rejected.

                                                                              Promise.allSettled([promise1, promise2, promise3])
  .then(results => {
    results.forEach((result) => console.log(result.status));
  });

                                                                              Promise.race()

                                                                              Returns a promise that resolves or rejects as soon as one of the promises in an iterable resolves or rejects.

                                                                              Promise.race([promise1, promise2, promise3])
  .then(value => {
    console.log(value);
  });

                                                                              Error Handling in Promises

                                                                              Errors in promises should be handled using the .catch() method to avoid uncaught rejections.

                                                                              myPromise
  .then(value => { /* handle resolved value */ })
  .catch(error => { /* handle rejection */ });

                                                                              Chaining Promises

                                                                              Promises can be chained, where .then() returns a new promise.

                                                                              myPromise
  .then(firstValue => { /* process and return new value */ })
  .then(secondValue => { /* handle second value */ })
  .catch(error => { /* handle any error that occurred in above .then() calls */ });

                                                                              # Promise.any()

                                                                              Returns a promise that resolves as soon as any one of the promises in the iterable resolves. It rejects if all of the provided promises are rejected.

                                                                              Promise.any([promise1, promise2, promise3])
  .then(value => {
    console.log(value); // the value from the first resolved promise
  })
  .catch(error => {
    console.log(error); // AggregateError: All promises were rejected
  });

                                                                              Promise.any() is particularly useful when you have multiple asynchronous tasks and want to proceed as soon as the first one completes successfully.

                                                                              Promise.any() enhances the versatility of Promise handling in JavaScript, providing a way to respond to the first success in a group of asynchronous operations.

                                                                              # Async/Await

                                                                              async/await simplifies working with promises, allowing for a more synchronous-looking code style.

                                                                              async function myAsyncFunction() {
    try {
        const value = await myPromise;
        // handle value
    } catch (error) {
        // handle error
    }
}

                                                                              await can only be used inside async functions and pauses the function execution until the promise is resolved.

                                                                              # Promise

                                                                              # Promise Constructor:

                                                                              • Usage: new Promise(executorFunction)
                                                                              • Description: Creates a new Promise object. The executorFunction is executed immediately by the Promise implementation, and it receives two functions as parameters: resolve and reject. The executorFunction is responsible for doing some asynchronous work and then calling the resolve or reject function to indicate that the work is done.

                                                                              Example:

                                                                              const myPromise = new Promise((resolve, reject) => {
  // Asynchronous operation here
  if (/* operation successful */) {
    resolve('Success!');
  } else {
    reject('Failure.');
  }
});

                                                                              # then()

                                                                              • Usage: promise.then(onFulfilled, onRejected) Description: Adds fulfillment and rejection handlers for the promise, and returns a new promise resolving to the return value of the called handler.

                                                                              • Example:

                                                                              myPromise.then(
  value => { console.log(value); }, // onFulfilled
  error => { console.error(error); } // onRejected
);

                                                                              # catch()

                                                                              • Usage: promise.catch(onRejected)
                                                                              • Description: Adds a rejection handler. It's shorthand for then(null, onRejected) and is typically used for error handling.
                                                                              • Example:
                                                                              myPromise.catch(
  error => { console.error(error); }
);

                                                                              # finally()

                                                                              • Usage: promise.finally(onFinally)

                                                                              • Description: Adds a handler that is called when the promise is settled, either fulfilled or rejected. This is useful for cleanup actions.

                                                                              • Example:

                                                                              myPromise.finally(() => {
  console.log('Completed, regardless of outcome.');
});

                                                                              # Promise.all()

                                                                              • Usage: Promise.all(iterable)
                                                                              • Description: Takes an iterable of promises and returns a single promise that resolves when all of the promises in the iterable have resolved, or rejects with the reason of the first promise that rejects.
                                                                              • Example:
                                                                              const promise1 = Promise.resolve(3);
const promise2 = 42;
const promise3 = new Promise((resolve, reject) => {
  setTimeout(resolve, 100, 'foo');
});

Promise.all([promise1, promise2, promise3]).then(values => {
  console.log(values); // [3, 42, "foo"]
});

                                                                              # Promise.race()

                                                                              • Usage: Promise.race(iterable)
                                                                              • Description: Returns a promise that resolves or rejects as soon as one of the promises in the iterable resolves or rejects, with the value or reason from that promise.
                                                                              • Example:
                                                                              const p1 = new Promise((resolve, reject) => { 
  setTimeout(resolve, 500, 'one'); 
});
const p2 = new Promise((resolve, reject) => { 
  setTimeout(resolve, 100, 'two'); 
});

Promise.race([p1, p2]).then(value => {
  console.log(value); // "two" - p2 is faster
});

                                                                              # Promise.allSettled()

                                                                              • Usage: Promise.allSettled(iterable)

                                                                              • Description: Returns a promise that resolves after all of the given promises have either resolved or rejected, with an array of objects that each describe the outcome of each promise.

                                                                              • Example:

                                                                              const p1 = Promise.resolve(3);
const p2 = new Promise((resolve, reject) => setTimeout(reject, 100, 'foo'));
const p3 = Promise.resolve(42);

Promise.allSettled([p1, p2, p3])
  .then(results => results.forEach((result) => console.log(result.status)));

// Output might be
// "fulfilled"
// "rejected"
// "fulfilled"

                                                                              # Promise.resolve()

                                                                              • Usage: Promise.resolve(value)
                                                                              • Description: Returns a Promise object that is resolved with the given value. If the value is a promise, that promise is returned; if the value is a thenable (i.e., has a "then" method), the returned promise will "follow" that thenable, adopting its eventual state; otherwise the returned promise will be fulfilled with the value.
                                                                              • Example:
                                                                              const resolvedPromise = Promise.resolve(42);
resolvedPromise.then(value => console.log(value)); // 42

                                                                              # Promise.reject()

                                                                              • Usage: Promise.reject(reason)
                                                                              • Description: Returns a Promise object that is rejected with the given reason.
                                                                              • Example:
                                                                              const rejectedPromise = Promise.reject(new Error('Fail'));
rejectedPromise.catch(error => console.error(error)); // Error: Fail
