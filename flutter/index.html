# runApp

There are 2 types of widges in the flutter.

  • Stateless widgets
  • If a widget doesn't do anything its Stateless widget.
  • Static in nature
  • Don't store any real-time data
  • Stateful widgets
  • If a widget does anything then it is Stateful widget.
  • Dynamic in nature.
